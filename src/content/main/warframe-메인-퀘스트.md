#### 메인 퀘스트 진행 순서
- 다른 퀘스트는 몰라도 메인 퀘스트는 꼭 클리어 하되 약간의 스펙이 필요하긴 해서 어느정도 성장 후에 진행하는걸 권장함

- 대부분의 메인 퀘스트는 진행중 다른 플레이가 불가능하고 그 중 시네마틱 퀘스트는 궤를 달리하는 플레이 시간을 자랑하기에 필수로 할거 하고 진행하자

- 언급이 안된 사이드 퀘스트는 알아서 클리어하셈 메인 퀘스트랑 연관이 없기에 급한건 아님 (단, "최초의 감염자" 퀘스트는 메인 퀘스트랑 연관이 있으니 이것 한정으로 빠른 클리어를 권장함)

### 되도록이면 아래에 적은 순서대로 클리어를 권장

---

#### 1. 각성
- 특징 : 튜토리얼

#### 2. 보어의 전리품
- 선행 조건 : 각성 클리어 후 자동 시작

#### 3. 스승
- 선행 조건 : 보어의 전리품 클리어
- 특징 : 모드 시스템 해금

#### 4. 각성의 순간
- 선행 조건 : 수성 교차점 클리어
- 특징 : 히트 소드 설계도

#### 5. 아크윙
- 선행 조건 : 화성 교차점 클리어
- 특징 : 아크윙 오도나타 획득

#### 6. 도둑맞은 꿈
- 선행 조건 : 포보스 교차점 클리어
- 특징 : 화성 마루의 암시장 해금

#### 7. 낯선 신비
- 선행 조건 : 도둑맞은 꿈 클리어, 유로파 교차점 클리어,  세팔론 시마리스와 대화
- 특징 : 생츄어리 맹습 해금

#### 8. 나타
- 선행 조건 : 천왕성의 수중 타일셋에서 무작위로 등장하는 오큘리스트 발견
- 특징 : 엑실러스 어댑터 해금

#### 9. 두 번째 꿈 (시네마틱 퀘스트 1)
- 선행 조건 : 나타 클리어, 해왕성 교차점 클리어
- 특징 : 오퍼레이터 해금

#### 10. 내면의 전쟁 (시네마틱 퀘스트 2)
- 선행 조건 : 두번째 꿈 클리어, 마스터리 랭크 5 이상, 모든 교차점 클리어
- 보상 : 포커스 유파 해금

#### 11. 해로우의 사슬
- 선행 조건 : 내면의 전쟁 클리어, 보이드의 Mot 노드 클리어
- 특징 : 해로우 설계도 획득

#### 12. 배교 프롤로그
- 선행 조건 : 내면의 전쟁 클리어후 보상인 개인실 제작 완료, 해로우 사슬 클리어시 개인실에서 보라색 구체와 상호작용

#### 13. 희생 (시네마틱 퀘스트 3)
- 선행 조건 : 배교 프롤로그 클리어
- 특징 : 엑스칼리버 움브라 획득

#### 14. 키메라 프롤로그
- 선행 조건 : 희생 클리어, 개인실에서 로터스 헤드기어 상호작용
- 특징 : 파라세시스 설계도 획득

#### 15. 밀려오는 물결
- 선행 조건 : 두번째 꿈 클리어, 레일잭 세팔론 제작, 정박소에서 활성화
- 특징 : 레일잭 해금

#### 16. 에라
- 선행 조건 : 개인실에 있는 로터스 헬멧 상호작용
- 특징 : 세두 설계도 획득

#### 17. 새로운 전쟁 (시네마틱 퀘스트 4)
- 선행 조건 : 레일잭 보유, 네크라메크 보유, 전쟁의 서막 (키메라-프롤로그, 에라, 창조주) 클리어
- 특징 : 드리프터 해금

#### 18. 자리만의 천사들
- 선행 조건 : 새로운 전쟁 클리어
- 특징 : 자리만 해금

#### 19. 베일 파괴자
- 선행 조건 : 새로운 전쟁 클리어

#### 20. 두비리의 역설 (시네마틱 퀘스트 5)
- 선행 조건 : 천왕성 교차점 클리어, 코덱스에서 시작
- 특징 : 두비리 해금

#### 21. 벽속의 속삭임 (시네마틱 퀘스트 6)
- 선행 조건 : 데이모스의 심장, 새로운 전쟁 클리어
- 특징 : 해부의 성역 해금

#### 22. 옥빛의 그림자 (시네마틱 퀘스트 7)
- 선행 조건 : 새로운 전쟁, 두비리의 역설, 천왕성 교차점 클리어
- 특징 : 제이드 설계도 획득

#### 23. 워프레임: 1999 (시네마틱 퀘스트 8)

##### 23-1. 연꽃을 먹는 자
- 선행 조건: 벽 속의 속삭임 클리어

##### 23-2. 헥스
- 선행 조건: 연꽃을 먹는 자 클리어, 메일함 확인
- 특징: 홀바니아 해금

##### 23-3. 헥스 피날레
- 선행 조건: 헥스의 평판 3단계, 모든 헥스 멤버 호감도 호감 단계

#### 24. 옛 평화 (시네마틱 퀘스트 9)
- 선행 조건 : 연꽃을 먹는 자를 포함한 모든 메인 퀘스트 클리어
- 특징 : 퀘스트 한정 엑스칼리버 프라임 사용, 디센디아, 페리타 반란 해금
