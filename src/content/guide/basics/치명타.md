### 워프레임 피해유형에 대해 알려드립니다
#### 치명타 편

워프레임식 치명타...얘는 좀 복잡함  
치명타 확률과 치명타 배수가 복합적으로 계산되어서 뎀지가 나오기 때문  
말고도 계산식이 그지같음

우선 사진을 먼저 보자

기본적으로 치명타가 터지면 노란색으로 데미지가 뜸

근데 100%을 초과하는 확률로 치명타가 터지면 주황크리, 200%를 초과하면 빨간크리, 300%를 초과하면 빨간크리에 맨 뒤에 ! 가 하나씩 추가됨

Q. 100%를 초과하는 치명타가 뭔 소리냐?

A. 예시로 치명타 모딩을 해서 치확이 180%가 되었다고 치자  
일단 노랑크리는 확정임 그리고 나머지 80% 확률로 주크가 터지냐 마냐가 됨

Q. 그럼 주크랑 빨크는 뎀지도 다름?

A. 다름 각 치명타 단계마다 배수가 1씩 늘어남

Q. 치명타배수는 뭐임?

A. 치명타 데미지를 결정하는 거임  
근데 계산식이 있음 ~~문제의 계산식~~

<span style="color:#9bbcff; font-style:italic;">
기본 치명타 배수 = 치명타 단계 × (모딩을 포함한 치명타 배수 - 1) + 1
</span>

이건데 딱봐도 복잡함 쉽게 설명 간다

기본데미지 100에 치명타배수가 3x인 무기로 노랑크리 터뜨리면 300데미지가 들어간다

Q. 왜 300이냐? 200아님?

A. 예시로 든 무기의 치명타 배수가 몇배냐? 3배잖아 기본치명타는 3배가 되는거임 그래서 300임  
노랑크리라고 무조건 기본데미지의 2배가 아닌거임 무기 속성창에 치명타 배수 이게 치명타 데미지를 결정함

Q. 그럼 모딩을 포함한 치명타 배수는 뭐임?

A. 다시 치명타배수 계산식을 보자

<span style="color:#9bbcff; font-style:italic;">
기본 치명타 배수 = 치명타 단계 × (모딩을 포함한 치명타 배수 - 1) + 1
</span>

아래사진의 프타크를 기준으로 설명한다

<p>
  <img
    src="/images/guide/primed-target-cracker.png"
    alt="프라임드 타겟 크래커"
    style="width:100%; max-width:220px; height:auto;"
  />
</p>
<span style="color:#9bbcff; font-style:italic;">
기본 치명타 배수 = 3  
모딩 배수 = +110% = 1.1  
무기데미지 = 100
</span>

식을 풀어서 계산넣으면

<span style="color:#9bbcff; font-style:italic;">
= (3 - 1) (1 + 1.1) +1<br />
= (2) (2.1) + 1<br />
= 4.2 + 1<br />
= 5.2
</span>

그럼 모딩을 포함한 노랑크리 데미지는  
100(무기데미지) X 5.2(모딩을 포함한 배수) = 520이 되는거

Q. 그럼 주황크리는 노랑크리의 두배인 1040임?

A. 아니다 좀 다르다

위의 계산식을 다시 보자

<span style="color:#9bbcff; font-style:italic;">
= (3 - 1) (1 + 1.1) +1<br />
= (2) (2.1) + 1<br />
= 4.2 + 1<br />
= 5.2
</span>

여기서 빠진게 하나 있다  
바로 치명타 단계 계산식  
노랑크리는 1단계 치명타라 생략되었는데 실제로는 아래의 계산이 더해진다

<span style="color:#9bbcff; font-style:italic;">
= 1 (5.2 - 1) + 1<br />
= 1 (4.2) + 1<br />
= 4.2 + 1<br />
= 5.2
</span>

인거라 사실상 노랑크리에선 의미없는 계산식인거임

근데 주황크리에선

<span style="color:#9bbcff; font-style:italic;">
= 2 (5.2 - 1) + 1<br />
= 2 (4.2) + 1<br />
= 8.2 + 1<br />
= 9.2<br />
= 920
</span>

으로 계산이 달라짐 ㅇㅋ?

그럼 빨크에선?

<span style="color:#9bbcff; font-style:italic;">
= 3 (5.2 - 1) + 1<br />
= 3 (4.2) + 1<br />
= 12.2 + 1<br />
= 13.2<br />
= 1320
</span>

이 되는거임

이런식으로 워프레임식 치명타 계산은 좀 뭐같음

사실 몰라도 되는 정보이긴 한데 그래도 알아두면 치명타 배수관련해서 좀 알기 쉬워질거임
